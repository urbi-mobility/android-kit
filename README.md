# Urbi Android Kit

`Urbi-Kit` is a Kotlin Android library that provides general functionality used in most applications, including Urbi!

When cloning this repository, make sure to initialize the submodules:

```bash
# Clone with submodules
```

After cloning with submodules:

```bash
./gradlew clean build
```
